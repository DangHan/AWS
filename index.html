Tóm tắt các kiến thức cốt lõi về dịch vụ máy chủ ảo Elastic Compute Cloud (EC2) của Amazon Web Services.

EC2 là gì?

Amazon Elastic Compute Cloud (EC2) là một dịch vụ web cung cấp năng lượng tính toán có thể thay đổi quy mô (resisable) trong đám mây. Nó được thiết kế để giúp việc tính toán quy mô web dễ dàng hơn cho các nhà phát triển.

  • Giảm thời gian khởi động máy chủ mới xuống còn vài phút.
  • Tăng hoặc giảm dung lượng ngay lập tức khi nhu cầu thay đổi.
  • Chỉ trả tiền cho những gì bạn thực sự sử dụng.

Các thành phần chính:

AMI

Template hệ điều hành & ứng dụng.

Instance Type

Cấu hình CPU, RAM, Network.

EBS

Ổ cứng lưu trữ dữ liệu (Block Storage).

Security Group

Tường lửa ảo kiểm soát traffic.

Các Họ Instance Phổ Biến

T

General Purpose

Cân bằng giữa CPU, bộ nhớ và mạng. Phù hợp cho web servers, code repositories.

Ví dụ: t2, t3, m5
C

Compute Optimized

Hiệu suất CPU cao. Phù hợp cho xử lý video, game servers, batch processing.

Ví dụ: c5, c6g
R

Memory Optimized

Dung lượng RAM cực lớn. Phù hợp cho in-memory databases (Redis), big data.

Ví dụ: r5, r6g

Các Hình Thức Thanh Toán (Purchasing Options)

On-Demand

Trả tiền theo giây/giờ. Không cam kết. Đắt nhất nhưng linh hoạt nhất.

Reserved Instances

Cam kết 1-3 năm. Giảm tới 75% chi phí so với On-Demand.

Spot Instances

Tận dụng dung lượng dư thừa. Giảm tới 90%. Có thể bị AWS thu hồi bất cứ lúc nào.

Dedicated Hosts

Máy chủ vật lý dành riêng. Phù hợp cho các yêu cầu tuân thủ (Compliance).

Bảo Mật Trong EC2

Security Groups (Stateful)

Đóng vai trò là tường lửa ảo ở cấp độ Instance. Mặc định chặn tất cả Inbound traffic và cho phép tất cả Outbound traffic.

Key Pairs

Sử dụng public-private key để đăng nhập vào Instance một cách an toàn thay vì sử dụng mật khẩu truyền thống.

Elastic IP

Địa chỉ IPv4 tĩnh, công khai mà bạn có thể gán nhanh chóng từ Instance này sang Instance khác khi có lỗi xảy ra.
